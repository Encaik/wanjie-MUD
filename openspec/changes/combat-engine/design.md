## Context

基于新属性/核心值系统，实现统一的回合制战斗引擎。消费核心值，不依赖旧属性系统。

## Goals / Non-Goals

**Goals:** 回合制战斗 + 速度出手权 + 物攻/特攻分离 + 技能CD + 多种开场类型

**Non-Goals:** 不实现战斗UI、不接入功法系统（仅预留接口）

## Decisions

### 战斗开场类型

| 类型 | 速度修正 | 说明 |
|------|----------|------|
| encounter | 无 | 正常遭遇 |
| ambush | 攻击方先手1轮 | 伏击 |
| surprise | 攻击方速度×1.5 | 出其不意 |
| defense | 防御方物防+50%首轮 | 防守战 |

### 伤害公式 (宝可梦式)

```
Damage = floor(((2×level + 10) / 250 × (ATK / DEF) × power + 2) × random(0.85, 1.0) × typeModifier)

- 物攻: ATK = physicalATK, DEF = physicalDEF
- 特攻: ATK = specialATK, DEF = specialDEF
```

### 战斗流程

```
1. 确定开场类型 → 计算出手顺序
2. 回合循环:
   a. 按速度排序活着的单位
   b. 当前单位随机选择未冷却技能
   c. 计算伤害 → 扣HP → 检查死亡
   d. 技能进入冷却 (time-based)
   e. 下一单位
3. 一方全灭 → 结束
```

### 技能系统

技能 = 基础威力(power) + 武器加成(bonus/penalty)。武器修正写在技能信息中。

### 战斗模式

- **手动**: 用户每回合选择技能
- **自动**: 自动选择冷却完毕中威力最高的技能

### 冷却系统

冷却 = 技能CD秒数 / 角色速度。速度越快冷却越短。
