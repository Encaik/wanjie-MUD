# 装备管理

## 概述

装备管理页面（`EquipManagePage`）合并旧装备、功法、技能三个独立页面为一个统一的槽位装备管理界面。

## 功能需求

### FR-1: 槽位视图

- 左侧面板展示三类槽位图标：
  - 装备槽位组（武器/头盔/身体/腿/脚 — 5个槽位，按 RPG 身体部位排列）
  - 功法槽位组（3个槽位）
  - 技能槽位组（6个槽位）
- 每个槽位显示：
  - 槽位名称
  - 当前装备物品的图标/名称 + 稀有度颜色
  - 空槽位显示 "+" 占位

### FR-2: 物品选择列表

- 右侧面板以分类 Tab 组织：
  - 装备 | 功法 | 技能
- 每个 Tab 显示背包中对应品类、未装备的物品列表
- 使用 `ItemCard` 组件渲染每个物品
- 点击物品装备到当前选中的槽位

### FR-3: 装备/卸载交互

- 点击槽位 → 右侧切换到对应分类列表，槽位高亮
- 点击列表中的物品 → 装备到高亮槽位（调用 `equipItem`）
- 点击已装备的物品 → 弹出确认对话框 → 卸载（调用 `unequipItem`）
- 装备/卸载结果以 message 反馈

### FR-4: 获取已装备物品

- 通过 `hook` 从 `protagonist.slots` + `protagonist.items` 读取
- 使用 `resolveItem` 获取合并后的物品信息用于渲染

## 路由

- 路径: `/game/equip-manage`
- 导航位置: "更多" → 武备组 → 装备管理
