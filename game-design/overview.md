# 游戏全景

## 设计原则

- **核心 Fantasy**: "穿越万界，在不同世界类型中修行成长"。世界系统是整个游戏的骨架
- **世界优先**: 玩家先选择世界，再选择角色（2026-06-09 `world-first-flow` 变更）
- **确定性生成**: 相同 seed 永远生成相同结果，前端只负责展示（计算在后端 API）
- **渐进式难度**: 飞升次数越多，可选择的世界难度越高

## 核心概念

- **万界**: 8 种不同世界类型（修仙、仙侠、高武、武侠、科技、魔幻、异能、末世），每种有独立的力量体系、境界名称和资源系统
- **化身**: 玩家在世界中选择的角色，由 seed 确定性生成，拥有属性、种族、天赋
- **飞升**: 玩家在某个世界完成修行后穿越到新世界，继承部分能力
- **Mod**: 通过注册中心（Registry）扩展世界类型、种族、属性等内容

## 核心循环

```
冒险 ──→ 战斗 ──→ 成长 ──→ 飞升
 │                    │
 │    ┌───────────────┘
 │    ▼
 └── 探索新区域 ←── 境界突破
```

1. **冒险**: 玩家在机缘系统中探索，触发随机事件，遭遇敌人
2. **战斗**: 回合制战斗，支持手动决策和自动战斗
3. **成长**: 通过修炼获得经验、提升等级、突破境界
4. **飞升**: 达到当前世界上限后飞升到更高难度世界，从头开始但保留部分收益

## 玩家旅程

```
                   启动页
                     │
                     ▼
              ┌──────────────┐
              │  万象星盘    │   ← 选择世界（8 种类型）
              │ (世界选择)   │
              └──────┬───────┘
                     │
                     ▼
              ┌──────────────┐
              │  命运之契    │   ← 从 8 个角色模板中选择
              │ (角色选择)   │
              └──────┬───────┘
                     │
                     ▼
              ┌──────────────┐
              │  宿命之章    │   ← 阅读背景故事，确认进入
              │ (背景故事)   │
              └──────┬───────┘
                     │
                     ▼
              ┌──────────────┐
              │  万界修行    │   ← 核心游戏循环
              │ (主游戏)     │     ├── 修炼/炼器/炼丹
              │              │     ├── 冒险/战斗
              │              │     ├── 商店/交易
              │              │     └── 势力/成就
              └──────┬───────┘
                     │
                     ▼
              ┌──────────────┐
              │  飞升穿越    │   ← 达到世界上限后穿越到新世界
              └──────────────┘
```

## 游戏阶段

| 阶段 | 内容 | 状态 |
|------|------|------|
| **选择期** | 世界选择 → 角色选择 → 背景故事 | ✅ 已重构 |
| **初期** | 修炼入门、低等级冒险、基础装备 | ⏳ 未重构 |
| **中期** | 境界突破、功法收集、势力加入 | ⏳ 未重构 |
| **后期** | 高难度秘境、装备打造、飞升准备 | ⏳ 未重构 |
| **终局** | 飞升穿越、多周目、排行榜 | ⏳ 未重构 |
