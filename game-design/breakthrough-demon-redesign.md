# 突破与心魔系统重设计

> 版本：v1.0  
> 日期：2026-06-18  
> 状态：提案阶段  
> 关联：`game-design/overview.md`、`game-design/characters.md`

---

## 一、问题诊断

### 1.1 当前突破系统的核心问题

| 问题 | 描述 | 严重度 |
|------|------|--------|
| **纯概率判定，零玩家操作** | 突破就是 `Math.random() < rate`，玩家除了堆幸运和吃丹药外没有任何交互手段 | 🔴 高 |
| **突破=升一级+随机加点属性** | 突破成功后没有任何仪式感，没有"我突破了"的爽感 | 🔴 高 |
| **与世界观属性脱节** | 8种世界有不同的属性体系（V3的11维CoreStat），但突破计算只用到了flat的幸运值 | 🟡 中 |
| **心魔与突破完全割裂** | 心魔只是修炼时随机触发的小弹窗，突破时完全没有心魔的参与 | 🔴 高 |
| **策略修炼缺乏深度** | 三种策略（稳健/激进/顿悟）只是灵石消耗和倍率的乘法，没有真正的决策空间 | 🟡 中 |
| **渡劫仅存在于Dialog中** | 渡劫是点击"迎接考验"→看结果，没有阶段感、没有紧张感 | 🟡 中 |

### 1.2 当前心魔系统的核心问题

| 问题 | 描述 | 严重度 |
|------|------|------|
| **5种固定心魔事件** | 贪念/恐惧/傲慢/悔恨/疑惑，随机出一个，选一个选项就行 | 🔴 高 |
| **没有"心魔养成"概念** | 心魔不会成长、不会进化、不会与玩家建立长期关系 | 🟡 中 |
| **选择深度不足** | 每个心魔的3个选项本质都是：成功率高中低对应奖励大中小 | 🟡 中 |
| **心智状态只有数字** | stability/karma/demonChance 只是三个数字，没有可视化反馈 | 🟡 中 |
| **世界观对心魔无影响** | 不同世界的"心魔"应该有不同形态——科技世界的心魔是AI失控，魔幻世界的是恶魔低语 | 🟡 中 |

### 1.3 根本原因

**突破和心魔被设计为两个独立系统，但实际上它们应该是同一条"修炼→突破→心魔考验→蜕变"叙事链上的不同环节。** 修炼积累经验是"量变"，突破是"质变"，心魔是质变前的"考验"。当前架构把它们拆成了三个互不关联的按钮。

## 二、设计目标

### 2.1 核心设计理念

> **突破不是概率，而是一场你可以操作、可以决策、有紧张感的"心魔挑战"。**

将突破和心魔融合为一个完整的"心魔突破战"系统（Demon Breakthrough Battle），让玩家在面对心魔时有真正的操作空间，突破成功与否取决于玩家的决策和属性，而非单纯的随机数。

### 2.2 四项设计原则

1. **操作感（Player Agency）**: 突破过程中玩家有多个决策点，每个决策影响结果
2. **世界观共鸣（World Resonance）**: 不同世界的属性、种族、天赋影响心魔形态和突破策略
3. **循序渐进（Progressive Depth）**: 低等级突破简单直接，高等级突破层次丰富
4. **仪式感（Ritual & Spectacle）**: UI/UX为突破设计专属的"心魔战场"界面，而非简单弹窗

### 2.3 成功指标

- 突破从"点击按钮看结果"变为至少 **3个阶段的交互体验**
- 心魔从5种固定事件扩展为 **世界观×流派组合的动态生成系统**
- 玩家属性（V3 CoreStats）**直接参与**心魔挑战的计算
- UI重新设计为全屏沉浸式"心魔战场"

## 三、新系统架构

### 3.1 突破流程重新设计

```
旧流程（当前）：
修炼攒经验 → 经验满了 → 点击修炼 → 概率判定 → 成功/失败

新流程（改进后）：
修炼攒经验 → 经验满了 → 点击"突破" → 进入心魔战场
    ├── 阶段一：心魔显形（属性检定）
    ├── 阶段二：心魔交锋（策略选择）
    ├── 阶段三：心魔炼化（意志对决）
    └── 结果：突破成功 / 突破失败但削弱心魔 / 被心魔反噬
```

### 3.2 心魔战场的三个阶段

#### 阶段一：心魔显形（属性检定阶段）

**玩法**：心魔显形时，系统根据玩家的**世界观属性、种族、天赋、业力值**动态生成心魔的"属性面板"。玩家需要用自己的11维CoreStats对抗心魔的对应属性。

```
┌─────────────────────────────────────────┐
│           心魔显形 · 属性检定              │
│                                         │
│  你的心魔是：【贪婪之煞】                   │
│  （世界观：修仙 → 心魔形态：贪婪煞气）       │
│                                         │
│  心魔属性：                               │
│  ████████░░  物理攻击  87  ←→  你的物理防御 62  │
│  ██████░░░░  特殊攻击  73  ←→  你的特殊防御 58  │
│  ████████░░  意志侵蚀  91  ←→  你的意志力   45  │
│                                         │
│  [检定结果] 物理防御不足！心魔造成 25 心境伤害  │
│  [使用丹药] [使用法宝] [硬抗]                │
└─────────────────────────────────────────┘
```

**技术要点**：
- 心魔属性 = f(玩家等级, 世界观难度, 业力值, 流派)
- 使用V3 CoreStats系统：physicalATK/specialATK 对应心魔的物理/法术攻击，willpower 对抗意志侵蚀
- 玩家可以使用修炼中积累的"心境护盾"（每次修炼成功+1层，失败-1层）来抵消伤害

#### 阶段二：心魔交锋（策略选择阶段）

**玩法**：心魔提出一个"诱惑"或"考验"，玩家从3种应对策略中选择。每种策略的成功率受**V3属性+流派技能+当前心境**影响。

```
┌─────────────────────────────────────────┐
│           心魔交锋 · 策略选择              │
│                                         │
│  "修仙之路漫长寂寥，放下执念，与我融为一体，   │
│   你将获得无上力量……只需放弃你的人性。"        │
│                                         │
│  ┌─────────────────────────────────┐    │
│  │ ◇ 坚守本心（意志检定）            │    │
│  │   成功率：68%                     │    │
│  │   成功：+30 心境稳定，+2 意志     │    │
│  │   失败：-20 心境，心魔强化        │    │
│  │   属性加成：意志+15, 灵根+10      │    │
│  └─────────────────────────────────┘    │
│  ┌─────────────────────────────────┐    │
│  │ ◇ 以智破魔（悟性检定）            │    │
│  │   成功率：82%                     │    │
│  │   成功：+15 心境稳定，+1 悟性     │    │
│  │   失败：-10 心境                  │    │
│  │   属性加成：悟性+12, 幸运+5       │    │
│  └─────────────────────────────────┘    │
│  ┌─────────────────────────────────┐    │
│  │ ◇ 剑斩心魔（体质检定）            │    │
│  │   成功率：45%                     │    │
│  │   成功：+50 心境稳定，+3 体质     │    │
│  │   失败：-40 心境，概率被反噬      │    │
│  │   属性加成：体质+20, 意志+5       │    │
│  └─────────────────────────────────┘    │
└─────────────────────────────────────────┘
```

**技术要点**：
- 成功率 = 基础成功率 + 对应CoreStat的值×权重 + 流派技能加成 + 心境稳定度加成
- 心魔的"诱惑文本"由世界观文本系统+心魔类型动态生成
- 魔修流派有独特的第4选项："吸收心魔"（高风险，永久提升心魔相关属性）

#### 阶段三：心魔炼化（意志对决阶段）

**玩法**：一个快速的"意志比拼"小游戏。玩家需要在有限时间内点击屏幕上随机出现的心魔弱点，每击中一个弱点积累"炼化进度"，同时心魔会攻击玩家的心境。进度条满 → 突破成功，心境被耗尽 → 突破失败。

```
┌─────────────────────────────────────────┐
│           心魔炼化 · 意志对决              │
│                                         │
│   心境护盾  ████████░░  72/100           │
│   炼化进度  ██████░░░░  58/100           │
│                                         │
│       ⚡ 弱点出现！ ⚡ 弱点出现！           │
│       [点击！]    [点击！]                │
│                                         │
│         （动态弱点球在界面游走）            │
│                                         │
│  心魔攻击！心境 -8                         │
│  道具栏：[清心丹×3] [护魂符×1]            │
└─────────────────────────────────────────┘
```

**技术要点**：
- 这是一个简单的反应小游戏（类似"打地鼠"），持续15-30秒
- 弱点出现频率 = f(perception 感知属性, speed 速度属性)
- 每次击中伤害 = f(physicalATK 或 specialATK，取决于流派)
- 心境护盾 = f(修炼中积累的层数 + willpower意志力)
- 玩家可以在战斗中**使用丹药/法宝**恢复心境或造成额外伤害
- 低等级（1-20级）此阶段自动跳过，或大幅简化

### 3.3 心魔生成系统（Demon Forge）

心魔不再从5个固定事件中随机选择，而是动态生成：

```
心魔 = 世界观模板 × 心魔类型 × 玩家数据 × 随机种子
```

#### 心魔属性（Demon Attributes）

| 属性 | 说明 | 来源 |
|------|------|------|
| 心魔名称 | 如"贪婪之煞"、"恐惧梦魇"、"傲慢AI" | 世界观心魔名池 + 心魔类型 |
| 心魔形态 | 视觉呈现（修仙→煞气/仙侠→剑魔/科技→AI幻象/魔幻→恶魔） | 世界观视觉配置 |
| 物理攻击 | 对心境的物理性冲击 | 基于玩家等级 × 世界观难度系数 |
| 特殊攻击 | 对心境的精神/法术冲击 | 基于玩家等级 × 世界观specialATK倍率 |
| 意志侵蚀 | 对willpower的直接考验 | 基于玩家的karma业力值（负业力→更高侵蚀） |
| 诱惑文本 | 心魔提出的"交易" | 世界观文本系统 + 心魔类型 |
| 弱点类型 | 物理/法术/意志（决定哪种攻击最有效） | 随机 + 流派影响 |

#### 心魔类型池（按世界观分类）

| 世界观 | 心魔主题 | 示例心魔 |
|--------|---------|---------|
| 修仙 | 七情六欲、执念 | 贪念之煞、情劫心魔、道心之魔 |
| 仙侠 | 剑道执念、杀业 | 剑意反噬、杀业心魔、迷茫剑心 |
| 高武 | 武道瓶颈、血脉 | 血脉暴走、武道迷茫、战意失控 |
| 科技 | 系统故障、伦理危机 | AI觉醒、基因崩溃、意识碎片 |
| 魔幻 | 恶魔契约、元素失控 | 恶魔低语、元素暴走、魔力反噬 |
| 异能 | 能力失控、人格分裂 | 异能暴走、人格镜像、基因记忆 |
| 武侠 | 心法反噬、恩怨情仇 | 走火入魔、仇恨心魔、武道迷茫 |
| 末世 | 变异恐惧、生存焦虑 | 丧尸幻象、辐射幻觉、孤寂疯狂 |

#### 玩家数据对心魔的影响

| 玩家因素 | 对心魔的影响 |
|----------|-------------|
| 高willpower（意志力） | 意志侵蚀降低，阶段三心境护盾增强 |
| 高perception（感知） | 阶段三弱点出现频率提升，更容易击中 |
| 高intelligence（智力） | 阶段二策略成功率提升，能看到更多信息 |
| 负karma（恶业） | 心魔攻击力增强，可能出现"业力心魔"变体 |
| 魔修流派 | 心魔强度+30%，但成功后吸收心魔获得额外属性 |
| 丹修流派 | 阶段三可使用"丹药投掷"对心魔造成伤害 |
| 体修流派 | 心境护盾上限+30%，阶段三更耐打 |
| 种族 | 不同种族对特定心魔类型有抗性（如龙族抗煞气） |

## 四、世界属性 → 心魔/突破的影响映射

### 4.1 V3 CoreStats 在两系统中的角色

```
CoreStat (11维)
├── maxHp          → 心魔战场中可承受的最大心境伤害
├── physicalATK    → 阶段三物理攻击弱点的伤害值
├── specialATK     → 阶段三法术攻击弱点的伤害值
├── physicalDEF    → 阶段一心魔物理攻击的减免
├── specialDEF     → 阶段一心魔法术攻击的减免
├── speed          → 阶段三弱点刷新速度 / 玩家可点击窗口
├── intelligence   → 阶段二策略成功率加成 / 可预见的选项结果
├── willpower      → 阶段一意志侵蚀抵抗 / 阶段三心境护盾基础值
├── lifespan       → 突破失败时的寿命消耗（特殊机制）
├── perception     → 阶段三弱点发现率 / 隐藏"完美炼化"机会
└── specialResourceCap → 使用法宝/特殊技能消耗的上限
```

### 4.2 各世界观属性的差异化体现

| 世界观 | 属性特点 | 对突破的影响 |
|--------|---------|-------------|
| 修仙 | 灵根(primary_spiritual)导向 | 心魔以法术攻击为主，specialATK/DEF权重高 |
| 高武 | 体质(primary_physical)导向 | 心魔以物理攻击为主，physicalATK/DEF权重高 |
| 科技 | 智力(intelligence)导向 | 阶段二有额外"数据分析"选项，intelligence权重+50% |
| 魔幻 | 特殊攻击导向 | 心魔附带元素效果（火心魔造成持续伤害） |
| 异能 | 速度(speed)导向 | 阶段三持续时间更短但弱点更多 |
| 仙侠 | 灵根+意志 | 阶段二有"剑心通明"专属策略 |
| 武侠 | 体质+意志 | 心魔强度较低但失败惩罚更重 |
| 末世 | 全属性波动大 | 心魔属性有随机变异，每次突破都不同 |

### 4.3 种族与天赋的心魔交互

- **人族**：通用适应性，无特殊加成也无特殊惩罚
- **龙族**（如果注册）：物理攻击型心魔伤害-20%，但意志侵蚀+20%（龙族傲慢）
- **魔族**（如果注册）：可吸收心魔获得双倍属性，但心魔强度+50%
- **天赋"心如磐石"**：心境护盾+50%
- **天赋"灵台清明"**：阶段二能看到每个选项的确切成功率

## 五、UI/UX 重新设计

### 5.1 心魔战场界面（全屏沉浸式）

不弹Dialog，而是展开一个**全屏心魔战场**。当前游戏界面渐隐，心魔战场从中心展开：

```
┌─────────────────────────────────────────────────────┐
│                                                     │
│              ╔═══════════════════╗                   │
│              ║   ⚡ 心魔战场 ⚡   ║                   │
│              ╚═══════════════════╝                   │
│                                                     │
│         （心魔形象区域 — 根据世界观生成）               │
│                                                     │
│    ╔══════════════════════════════════════╗          │
│    ║  阶段一：属性检定                      ║          │
│    ║  心魔的煞气向你袭来……                  ║          │
│    ║  [属性对比条] [检定动画] [结果]        ║          │
│    ╚══════════════════════════════════════╝          │
│                                                     │
│    ╔══════════════════════════════════════╗          │
│    ║  阶段二：策略选择                      ║          │
│    ║  "放弃抵抗，与我融合吧……"              ║          │
│    ║  [选项A] [选项B] [选项C] [选项D魔修]    ║          │

│                                                     │
│    ╔══════════════════════════════════════╗          │
│    ║  阶段三：心魔炼化                      ║          │
│    ║  [心境护盾条] [炼化进度条]              ║          │

│    ╚══════════════════════════════════════╝          │
│                                                     │
└─────────────────────────────────────────────────────┘
```

### 5.2 视觉风格

- **心魔战场背景**：根据世界观动态生成（修仙→水墨幻境/科技→数字矩阵/魔幻→魔法阵）
- **心魔形象**：使用CSS动画+粒子效果，不同心魔类型有不同视觉
- **属性对比条**：使用`Progress`组件，颜色动态变化（优势绿/劣势红）
- **弱点球**：使用CSS关键帧动画，随机游走，带发光效果
- **突破成功特效**：全屏金色光芒+境界名称大字浮现+属性增长飘字动画

### 5.3 突破成功/失败的叙事反馈

**突破成功**：
```
天地异象！万道共鸣！
【金丹期·三重】突破成功！
体质 +3  灵根 +2  意志 +1
获得被动：金丹护体（物理防御永久+5%）
```

**突破失败（但削弱了心魔）**：
```
心魔暂时退散……
虽然未能突破，但你已经摸到了门槛。
下次突破时心魔强度 -20%
心境稳定度 +10（因为你坚持了下来）
```

**被心魔反噬**：
```
心魔趁虚而入！你被煞气侵蚀……
等级未变，但心境受损。
心境稳定度 -30，心魔下次强度 +30%
建议：打坐冥想/服用清心丹恢复后再尝试
```

## 六、突破阶段三的"心魔炼化"交互设计详解

阶段三是整个系统中"操作感"最强的部分，需要仔细设计。

### 6.1 核心玩法

一个限时（15-30秒，由speed属性决定）的点击游戏：

1. 圆形/球形的"心魔弱点"在心魔战场区域随机出现、移动、消失
2. 玩家点击弱点造成伤害，积累"炼化进度条"
3. 同时心魔持续攻击玩家的"心境护盾"
4. 进度条满 → 突破成功
5. 心境护盾归零 → 突破失败（但如果进度条超过50%，下次心魔强度降低）

### 6.2 弱点类型

| 弱点颜色 | 对应属性 | 伤害公式 | 出现概率 |
|---------|---------|---------|---------|
| 🔴 红色 | physicalATK | 伤害 = physicalATK × 1.0 × 暴击系数 | 40% |
| 🔵 蓝色 | specialATK | 伤害 = specialATK × 1.2 × 暴击系数 | 35% |
| 🟡 金色 | 暴击弱点 | 伤害 = (physicalATK + specialATK) × 2.0 | 5% |
| 🟢 绿色 | 恢复 | 恢复心境护盾 15点 | 10% |
| ⚫ 黑色 | 陷阱 | 点击后心境-10，但炼化进度+15 | 10% |

### 6.3 道具使用

玩家在阶段三可以使用准备阶段携带的道具（最多3个）：

| 道具 | 效果 |
|------|------|
| 清心丹 | 恢复30点心境护盾 |
| 破魔符 | 对所有当前弱点造成20点伤害 |
| 凝神香 | 5秒内弱点移动速度减半 |
| 护魂镜 | 5秒内心境护盾不减少 |
| 暴击丹 | 10秒内暴击率翻倍 |

### 6.4 难度曲线

| 等级区间 | 阶段三持续 | 弱点数量 | 心魔每秒伤害 | 道具槽位 |
|---------|-----------|---------|------------|---------|
| 1-10 | 跳过阶段三（简化） | - | - | - |
| 11-20 | 15秒 | 2-3个 | 3点/秒 | 1个 |
| 21-40 | 20秒 | 3-4个 | 5点/秒 | 2个 |
| 41-60 | 25秒 | 3-5个 | 8点/秒 | 2个 |
| 61-80 | 30秒 | 4-6个 | 10点/秒 | 3个 |
| 81-100 | 30秒 | 5-7个 | 12点/秒 | 3个 |

### 6.5 特殊机制

- **完美炼化**（perception ≥ 80时解锁）：5%概率出现"完美炼化"机会——在0.5秒内点击一个闪现的金色弱点，直接完成炼化
- **流派终结技**（流派等级 ≥ 5时解锁）：阶段三可以使用一次流派专属技能
  - 体修：金刚怒目——5秒内所有伤害×3
  - 剑修：一剑破魔——瞬间造成50点伤害
  - 法修：万法归一——弱点移动停止3秒
  - 丹修：丹火焚魔——持续10秒，每秒自动对最近的弱点造成伤害
  - 魔修：魔噬——消耗30心境护盾，瞬间造成80点伤害

### 9.2 关键API设计

```typescript
// 心魔生成
interface DemonBreakthrough {
  demon: GeneratedDemon;       // 动态生成的心魔
  phases: {
    attributeCheck: AttributeCheckPhase;
    strategyChoice: StrategyChoicePhase;
    refineBattle: RefineBattlePhase;
  };
  worldResonance: WorldResonanceData;  // 世界观差异数据
}

// 心魔属性
interface GeneratedDemon {
  id: string;
  name: string;               // 动态生成的心魔名称
  type: DemonType;            // 心魔类型枚举
  worldType: string;          // 所属世界观
  visualPreset: DemonVisualPreset;  // 视觉配置
  stats: {
    physicalAttack: number;   // 物理攻击（对应 physicalDEF）
    specialAttack: number;    // 特殊攻击（对应 specialDEF）
    willErosion: number;      // 意志侵蚀（对应 willpower）
  };
  temptation: string;         // 诱惑文本
  weakPointType: 'physical' | 'special' | 'balanced';  // 弱点类型
}

// 阶段一结果
interface AttributeCheckResult {
  passed: boolean;
  checks: {
    physical: { playerDEF: number; demonATK: number; damage: number };
    special: { playerDEF: number; demonATK: number; damage: number };
    will: { playerWill: number; demonErosion: number; damage: number };
  };
  totalMindDamage: number;  // 心境总伤害
  bonuses: string[];        // 种族/天赋/道具提供的检定加成
}

// 阶段三逻辑
interface RefineBattleResult {
  progress: number;           // 炼化进度 0-100
  mindShield: number;         // 剩余心境护盾 0-100
  weaknessHit: number;        // 击中弱点次数
  weaknessMissed: number;     // 错过的弱点
  perfectChanceUsed: boolean; // 是否触发了完美炼化
  itemsUsed: string[];        // 使用的道具ID列表
  finalSkillUsed: boolean;    // 是否使用了流派终结技
}
```

### 9.3 兼容性考虑

- V3 CoreStats 系统当前在 `core/types/types.ts` 中定义但尚未完全应用到业务逻辑
- 突破系统需要同时支持旧5属性（FlatStats）和新11维CoreStats
- 使用适配层：如果V3属性未初始化，从FlatStats推导CoreStats的近似值
- 旧的心魔触发机制（`useMentalState` + `checkDemonTrigger`）在新的突破流程上线前继续工作
- `MentalState` 数据结构扩展：新增 `mindShield: number`（心境护盾层数）和 `demonCodex: DemonMemory[]`（心魔图鉴）

### 9.4 测试策略

- 心魔生成的确定性：给定相同seed + 相同玩家数据 → 生成相同心魔
- 阶段一属性检定的边界：各种属性极值（0、999、负数）下的行为
- 阶段三的逻辑正确性：各种操作序列的期望结果
- UI组件的无障碍：键盘可操作（方向键选择选项、空格点击弱点）

## 十、与现有系统的对接

### 10.1 与修炼系统

- `executeCultivation()` 不再包含突破逻辑，只负责经验积累
- 经验满后，"修炼"按钮变为"冲击境界"按钮
- 策略修炼（稳健/激进/顿悟）影响进入心魔战场时的初始状态：
  - 稳健修炼：+10 心境护盾
  - 激进修炼：-5 心境护盾，+10% 弱点伤害
  - 顿悟修炼：+5% 完美炼化触发率

### 10.2 与闭关系统

- 闭关修炼触发突破时，自动进入心魔战场
- 闭关效果影响心魔战场：
  - 天人交感：跳过阶段一，直接进入阶段二
  - 大彻大悟：阶段二额外+15%成功率
  - 走火入魔：心魔强度+30%

### 10.3 与飞升系统

- 飞升前的最后一次突破（满级突破）使用特殊的"终极心魔"
- 终极心魔 = 玩家在该世界击败的所有心魔的聚合体
- 战胜终极心魔获得"心魔王"称号和额外飞升奖励

### 10.4 与流派系统

- 流派等级≥5解锁流派终结技（阶段三）
- 各流派在心魔战场中有不同的属性加成权重

## 十一、风险评估

| 风险 | 概率 | 影响 | 缓解措施 |
|------|------|------|---------|
| 阶段三交互太复杂，玩家不喜欢 | 中 | 中 | 低等级自动跳过；提供"自动炼化"选项（消耗额外资源跳过交互） |

| 心魔生成文本重复感 | 中 | 低 | 使用模板+变量组合，确保高变体数 |

## 十二、总结

这个重设计方案将**突破**和**心魔**从两个独立的"点按钮看结果"系统，融合为一个有操作感、有策略深度、有世界观共鸣的"心魔突破战"三阶段流程。核心改进：

1. **操作感**：阶段三的点击交互让突破不再是纯随机事件
2. **策略深度**：阶段二的策略选择受CoreStats+流派+世界观多重影响
3. **世界观共鸣**：不同世界的属性、心魔形态、交互视觉各不相同
4. **长期养成**：心魔图鉴、宿敌系统、心魔精华让每次突破都有意义
5. **仪式感**：全屏心魔战场、突破特效、叙事文本强化"突破"的体验

建议优先实现第一阶段（核心融合MVP），验证核心循环后再逐步增加交互和差异化内容。
