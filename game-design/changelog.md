# 设计变更日志

## 2026-06-16 - 新手引导 + 统计系统 + 任务面板重构（设计中）

- **改动**: 设计统一统计追踪系统（`core/statistics/`）、事件驱动分阶段新手引导（5 阶段 9 步骤）、Tab 式任务中心面板；统计触发从分散式改为 EventBus 事件驱动；初始物品归入引导体系
